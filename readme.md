# Návod na inštaláciu aplikácie
Pre spustenie tejto webovej aplikácie treba mať nainštalovaných **docker**. V adresári **docker** treba pomocou **docker compose** spustiť kontajnery narp.: ``` docker compose up ```.

Pri spustení sa vytvoria potrebné tabuľky v databáze a vytvorí sa používateľ **admin** s heslom **1234567890**. Tento používateľ ma práva administrátora webu. Jeho heslo je odporúčané zmeniť.

Web je prístupný na porte **80** (do prehliadača stačí len zadať ip adresu alebo localhost v prípade inštalácie na počítači)
